4. Description of Service

Full Disk Clean is a macOS utility application that scans your file system to analyze disk usage, visualize storage breakdown by category, and identify files that may be safely removed to reclaim disk space. The App provides the following functionality:

  • File System Scanning: Reads file metadata (names, sizes, paths, modification dates) to display disk usage information
  • Storage Visualization: Presents disk usage as interactive charts and categorized breakdowns
  • File Cleanup: Identifies reclaimable files and, at your explicit direction, moves selected files to the macOS Trash
  • Safety Classification: Categorizes identified items into safety tiers (Safe, Caution, Danger) to help you make informed decisions
